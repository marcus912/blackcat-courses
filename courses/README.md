# Courses

Source course materials (handbooks / textbooks) for the 26-week program. These are the *inputs* — weekly notes and homework that build on them live under [`weeks/`](../weeks/).

One folder per course; each holds the course's PDF(s) and any future supplements (slides, exercise files). A single course usually spans several weeks.

| Folder | Course | Topic |
|---|---|---|
| [`python-first-class-functions/`](./python-first-class-functions/) | Python 一級函式修煉手冊 | Python first-class functions |
| [`docker-engineering/`](./docker-engineering/) | Docker：從容器入門到規模化、安全與可觀測性的實戰工程 | Docker: containers → scale, security, observability |
| [`cloud-native-fullstack/`](./cloud-native-fullstack/) | 雲原生全棧實戰 — 從一個容器，到一座 AI 工廠 | Cloud-native full-stack: from a container to an AI factory |
| [`llm-complete-course/`](./llm-complete-course/) | 玄貓大型語言模型完整課程 | Complete large language model course |

> The overall schedule is in [`../CourceSchedule.pdf`](../CourceSchedule.pdf). Curriculum is subject to change — see [`docs/roadmap.md`](../docs/roadmap.md) for live per-week status.
